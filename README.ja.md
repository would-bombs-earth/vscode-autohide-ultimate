# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** は、VS Codeのサイドバーとパネル管理の究極のソリューションです。クリックに応じた自動表示切り替えだけでなく、遅延処理、アイドル時の非表示、環境認識などの高度な機能により、最も純粋で没入感のあるプログラミング環境を提供します。

---

## ✨ コア機能

![Demo](https://raw.githubusercontent.com/would-bombs-earth/vscode-autohide-ultimate/main/Images/Features/AutoHideSideBar.gif)

### 🖱️ インタラクティブな自動非表示
*   **サイドバー**: エディタをクリックすると、左側または右側のサイドバーを自動的に非表示にします。
*   **パネル**: ターミナル、出力、デバッグコンソールの表示を自動的に管理します。
*   **参照検索**: 参照への移動後、エディタをクリックすると検索ウィンドウを自動的にクリアします。

### 🛡️ デバッグモード連携
*   **環境認識**: デバッグ状態に応じて表示戦略を自動的に切り替えます。
*   **自動化**: デバッグ開始時にパネルを開き、終了時に元のクリーンな状態に戻します。

---

## 🏆 アルティメット版限定機能

### ⏱️ 精密な遅延非表示 (Hide Delay)
誤操作によるUIのちらつきを防ぎます。遅延時間（例：300ms）を設定することで、UIがより自然に消えるようになります。

### 🚫 インテリジェントな言語除外 (Exclusion List)
Markdownの執筆中にアウトラインを表示し続けたいですか？ `excludeLanguages` 設定により、特定の言語ファイルで自動非表示を無効にできます。

### 💤 アイドル時の自動非表示 (Idle Auto-Hide)
一定時間操作がない場合（例：10秒）、自動的にUIを整理し、思考を妨げないようにします。

### 🌑 フォーカス喪失時の非表示 (Focus-Lost Hide)
ブラウザに切り替えると、VS Codeは自動的に補助バーを格納します。戻ったときには、広々とした編集エリアがあなたを待っています。
