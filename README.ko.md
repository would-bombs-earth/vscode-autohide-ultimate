# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate**는 VS Code 사이드바 및 패널 관리의 궁극적인 솔루션입니다. 단순한 자동 숨김을 넘어 지연 처리, 유휴 상태 숨김, 환경 인식 등의 고급 기능을 통해 가장 순수하고 몰입감 있는 프로그래밍 환경을 제공합니다.

---

## ✨ 핵심 기능

![Demo](Images/Features/AutoHideSideBar.gif)

### 🖱️ 대화형 자동 숨김
*   **사이드바**: 에디터를 클릭하면 왼쪽 또는 오른쪽 사이드바를 자동으로 숨깁니다.
*   **패널**: 터미널, 출력, 디버그 콘솔의 가시성을 자동으로 관리합니다.
*   **참조 검색**: 참조 이동 후 에디터를 클릭하면 잔류 검색 창을 자동으로 정리합니다.

### 🛡️ 디버그 모드 연동
*   **환경 인식**: 디버깅 상태에 따라 표시 전략을 자동으로 전환합니다.
*   **자동화**: 디버깅 시작 시 패널을 열고, 종료 시 깨끗한 인터페이스로 복구합니다.

---

## 🏆 얼티밋 버전 전용 기능

### ⏱️ 정밀 지연 숨김 (Hide Delay)
오클릭으로 인한 UI 깜빡임을 방지합니다. 지연 시간(예: 300ms)을 설정하여 UI가 더 자연스럽고 매끄럽게 사라지도록 할 수 있습니다.

### 🚫 스마트 언어 제외 (Exclusion List)
Markdown 작성 시 아웃라인을 계속 보고 싶으신가요? `excludeLanguages` 설정을 통해 특정 언어 파일에서 자동 숨김 로직을 무시할 수 있습니다.

### 💤 유휴 상태 자동 숨김 (Idle Auto-Hide)
일정 시간 동안 조작이 없으면(예: 10초) 자동으로 UI를 정리하여 사고를 방해하지 않도록 합니다.

### 🌑 포커스 상실 시 숨김 (Focus-Lost Hide)
브라우저로 전환하면 VS Code가 자동으로 모든 보조 바를 접습니다. 다시 돌아오면 가장 넓은 편집 영역이 여러분을 맞이합니다.
