# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** 是 VS Code 側邊欄和面板管理的終極解決方案。它不僅能根據你的點擊自動切換 UI 顯示，還引入了延遲處理、閒置隱藏和環境感知等高級功能，為你打造最純粹、最沉浸的編程環境。

---

## ✨ 核心功能 (Core Features)

![Demo](https://raw.githubusercontent.com/would-bombs-earth/vscode-autohide-ultimate/main/Images/Features/AutoHideSideBar.gif)

### 🖱️ 交互式自動隱藏
*   **側邊欄 (SideBar)**: 點擊編輯器時自動隱藏左側或右側欄。
*   **底部面板 (Panel)**: 自動管理終端、輸出、調試控制台的可見性。
*   **引用搜索 (References)**: 在跳轉引用後點擊編輯器，自動清理殘留的搜索窗口。

### 🛡️ 調試模式聯動
*   **環境感知**: 自動根據是否處於調試狀態切換不同的顯示策略。
*   **入場/離場自動化**: 進入調試時打開調試面板，退出時恢復清爽界面。

---

## 🏆 終極版獨有功能 (Ultimate Pro Features)

### ⏱️ 精準延遲隱藏 (Hide Delay)
不再因為誤觸導致 UI 頻繁閃爍。你可以設置延遲時間（如 300ms），讓 UI 的消失更加自然、平滑。

### 🚫 智能語言排除 (Exclusion List)
在編寫 Markdown 時需要一直看著 Outline？在處理 JSON 時需要參考結構？
通過 `excludeLanguages` 設置，你可以讓特定語言的文件完全忽略自動隱藏邏輯。

### 💤 閒置自動隱藏 (Idle Auto-Hide)
如果你停止操作一段時間（如 10 秒），Auto Hide Ultimate 會自動為你清理桌面，讓你在思考時不受任何 UI 干擾。

### 🌑 失去焦點自動隱藏 (Focus-Lost Hide)
當你切換到瀏覽器查找資料時，VS Code 會自動收起所有輔助欄。當你切回 VS Code 時，迎接你的是最寬敞的編輯區域。
