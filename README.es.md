# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** es la solución definitiva para gestionar la barra lateral y el panel de VS Code. Ofrece un control de precisión sobre su espacio de trabajo, proporcionando un entorno libre de distracciones para el trabajo profundo.

---

## ✨ Funciones principales

### 🖱️ Ocultación automática interactiva
*   **Barra lateral**: Oculta automáticamente la barra lateral cuando hace clic en el editor.
*   **Panel**: Gestione sin problemas la visibilidad de la terminal, la salida y la consola de depuración.
*   **Referencias**: Borra automáticamente las ventanas de búsqueda de referencias después de la navegación.

### 🛡️ Integración de depuración
*   **Consciente del contexto**: Cambia los diseños de la interfaz de usuario automáticamente según el estado de depuración.
*   **Automatización**: Abre los paneles de depuración al iniciar y restaura la interfaz de usuario limpia al detener.

---

## 🏆 Funciones Ultimate Pro

### ⏱️ Retraso de ocultación de precisión
Evite el parpadeo de la interfaz de usuario causado por clics accidentales. Establezca un retraso (por ejemplo, 300 ms) para una transición más suave.

### 🚫 Exclusión inteligente de idiomas
¿Necesita el esquema visible para Markdown? Use `excludeLanguages` para omitir la ocultación automática para tipos de archivos específicos.

### 💤 Ocultación automática por inactividad
Auto Hide Ultimate puede limpiar automáticamente su espacio de trabajo después de un período de inactividad (por ejemplo, 10 segundos).

### 🌑 Ocultar al perder el foco
Cuando cambia a un navegador, VS Code guarda automáticamente todas las barras laterales, recibiéndolo de nuevo con un editor limpio y espacioso.
