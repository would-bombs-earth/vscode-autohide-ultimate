# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** è la soluzione definitiva per gestire la barra laterale e il pannello di VS Code. Offre un controllo di precisione sulla tua area di lavoro, fornendo un ambiente privo di distrazioni per il lavoro profondo.

---

## ✨ Funzionalità principali

![Demo](https://raw.githubusercontent.com/would-bombs-earth/vscode-autohide-ultimate/main/Images/Features/AutoHideSideBar.gif)

### 🖱️ Nascondi automaticamente interattivo
*   **Barra laterale**: Nasconde automaticamente la barra laterale quando si fa clic nell'editor.
*   **Pannello inferiore**: Gestisci senza problemi la visibilità di terminale, output e console di debug.
*   **Riferimenti**: Cancella automaticamente le finestre di ricerca dei riferimenti dopo la navigazione.

### 🛡️ Integrazione debug
*   **Consapevolezza del contesto**: Cambia automaticamente i layout dell'interfaccia utente in base allo stato del debug.
*   **Automazione**: Apre i pannelli di debug all'avvio e ripristina l'interfaccia pulita all'arresto.

---

## 🏆 Funzionalità Ultimate Pro

### ⏱️ Ritardo di occultamento di precisione
Evita lo sfarfallio dell'interfaccia utente causato da clic accidentali. Imposta un ritardo (es. 300 ms) per una transizione più fluida.

### 🚫 Esclusione intelligente della lingua
Hai bisogno dell'Outline visibile per Markdown? Usa `excludeLanguages` per bypassare l'occultamento automatico per tipi di file specifici.

### 💤 Occultamento automatico inattivo
Auto Hide Ultimate può pulire automaticamente la tua area di lavoro dopo un periodo di inattività (es. 10 secondi).

### 🌑 Nascondi alla perdita di focus
Quando passi a un browser, VS Code nasconde automaticamente tutte le barre laterali, accogliendoti di nuovo con un editor pulito e spazioso.
