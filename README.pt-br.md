# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** é a solução definitiva para gerenciar a barra lateral e o painel do VS Code. Ele oferece controle de precisão sobre seu espaço de trabalho, proporcionando um ambiente livre de distrações para o trabalho profundo.

---

## ✨ Recursos principais

### 🖱️ Ocultação automática interativa
*   **Barra lateral**: Oculta automaticamente a barra lateral quando você clica no editor.
*   **Painel inferior**: Gerencie perfeitamente a visibilidade do terminal, saída e console de depuração.
*   **Referências**: Limpa automaticamente as janelas de pesquisa de referências após a navegação.

### 🛡️ Integração de depuração
*   **Ciente do contexto**: Alterna layouts de interface do usuário automaticamente com base no estado de depuração.
*   **Automação**: Abre painéis de depuração ao iniciar e restaura a interface do usuário limpa ao parar.

---

## 🏆 Recursos Ultimate Pro

### ⏱️ Atraso de ocultação de precisão
Evite a oscilação da interface do usuário causada por cliques acidentais. Defina um atraso (ex: 300 ms) para uma transição mais suave.

### 🚫 Exclusão inteligente de idiomas
Precisa do Outline visível para Markdown? Use `excludeLanguages` para ignorar a ocultação automática para tipos de arquivos específicos.

### 💤 Ocultação automática em inatividade
O Auto Hide Ultimate pode limpar automaticamente seu espaço de trabalho após um período de inatividade (ex: 10 segundos).

### 🌑 Ocultar ao perder o foco
Quando você muda para um navegador, o VS Code guarda automaticamente todas as barras laterais, recebendo você de volta com um editor limpo e espaçoso.
