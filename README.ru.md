# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** — это окончательное решение для управления боковой панелью и панелью VS Code. Оно обеспечивает точный контроль над вашим рабочим пространством, создавая среду без отвлекающих факторов для глубокой работы.

---

## ✨ Основные функции

![Demo](Images/Features/AutoHideSideBar.gif)

### 🖱️ Интерактивное автоскрытие
*   **Боковая панель**: автоматически скрывает боковую панель при клике в редакторе.
*   **Нижняя панель**: легко управляет видимостью терминала, вывода и консоли отладки.
*   **Ссылки**: автоматически очищает окна поиска ссылок после навигации.

### 🛡️ Интеграция с отладкой
*   **Контекстная осведомленность**: автоматически переключает макеты интерфейса в зависимости от состояния отладки.
*   **Автоматизация**: открывает панели отладки при запуске и восстанавливает чистый интерфейс при остановке.

---

## 🏆 Функции версии Ultimate Pro

### ⏱️ Точная задержка скрытия
Избегайте мерцания интерфейса, вызванного случайными кликами. Установите задержку (например, 300 мс) для более плавного перехода.

### 🚫 Интеллектуальное исключение языков
Нужна структура для Markdown? Используйте `excludeLanguages`, чтобы обойти автоскрытие для определенных типов файлов.

### 💤 Автоскрытие при простое
Auto Hide Ultimate может автоматически очищать рабочее пространство после периода бездействия (например, 10 секунд).

### 🌑 Скрытие при потере фокуса
При переключении на браузер VS Code автоматически убирает все боковые панели, встречая вас чистым и просторным редактором по возвращении.
