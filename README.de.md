# 🚀 Auto Hide Ultimate


**Auto Hide Ultimate** ist die definitive Lösung für die Verwaltung der VS Code Seitenleiste und des Panels. Es bietet Präzisionssteuerung für Ihren Arbeitsbereich und ermöglicht eine ablenkungsfreie Umgebung für konzentriertes Arbeiten.

---

## ✨ Kernfunktionen

![Demo](Images/Features/AutoHideSideBar.gif)

### 🖱️ Interaktives automatisches Ausblenden
*   **Seitenleiste**: Blendet die Seitenleiste automatisch aus, wenn Sie in den Editor klicken.
*   **Panel**: Verwaltet nahtlos die Sichtbarkeit von Terminal, Ausgabe und Debug-Konsole.
*   **Referenzen**: Bereinigt automatisch Referenzsuchfenster nach der Navigation.

### 🛡️ Debug-Integration
*   **Umgebungsbewusst**: Schaltet UI-Layouts automatisch basierend auf dem Debug-Status um.
*   **Automatisierung**: Öffnet Debug-Panels beim Start und stellt die saubere UI beim Stoppen wieder her.

---

## 🏆 Ultimate Pro Funktionen

### ⏱️ Präzise Verzögerung (Hide Delay)
Vermeiden Sie UI-Flackern durch versehentliche Klicks. Stellen Sie eine Verzögerung (z. B. 300 ms) für einen sanfteren Übergang ein.

### 🚫 Intelligente Sprachausschlüsse (Exclusion List)
Benötigen Sie die Outline für Markdown? Nutzen Sie `excludeLanguages`, um das automatische Ausblenden für bestimmte Dateitypen zu umgehen.

### 💤 Automatisches Ausblenden bei Inaktivität
Auto Hide Ultimate bereinigt Ihren Arbeitsbereich automatisch nach einer Zeit der Inaktivität (z. B. 10 Sekunden).

### 🌑 Ausblenden bei Fokusverlust
Wenn Sie zum Browser wechseln, verstaut VS Code automatisch alle Seitenleisten und empfängt Sie bei der Rückkehr mit einem sauberen Editor.
