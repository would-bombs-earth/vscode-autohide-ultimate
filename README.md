# 🚀 Auto Hide Ultimate

**Auto Hide Ultimate** is the definitive solution for managing VS Code SideBar and Panel visibility. It offers precision control over your workspace, providing a distraction-free environment for deep work.

---

## ✨ Core Features

### 🖱️ Interactive Auto-Hide
*   **SideBar**: Automatically hide the SideBar when you click into the editor.
*   **Bottom Panel**: Seamlessly manage the visibility of Terminal, Output, and Debug Console.
*   **References**: Automatically clear reference search windows after navigation.

### 🛡️ Debug Integration
*   **Context Aware**: Switch UI layouts automatically based on debug state.
*   **Auto Entry/Exit**: Open debug panels on start and restore clean UI on stop.

---

## 🏆 Ultimate Pro Features

### ⏱️ Precision Hide Delay
Avoid UI flickering caused by accidental clicks. Set a delay (e.g., 300ms) for a smoother, more natural transition.

### 🚫 Smart Language Exclusion
Need the Outline visible for Markdown? Keeping the structure open for JSON? Use `excludeLanguages` to bypass auto-hide for specific file types.

### 💤 Idle Auto-Hide
Auto Hide Ultimate can automatically clean up your workspace after a period of inactivity (e.g., 10 seconds), keeping you focused.

### 🌑 Hide on Focus Lost
When you switch to a browser, VS Code automatically tucks away all sidebars, welcoming you back with a clean, spacious editor.

### 🔘 Master Toggle
Quickly enable or disable all features with a single command or status bar toggle.

---

## ⚙️ Quick Configuration

| Setting | Description | Default |
| :--- | :--- | :--- |
| `autoHidePlus.enabled` | Global master switch | `true` |
| `autoHidePlus.hideDelay` | Delay before hiding (ms) | `0` |
| `autoHidePlus.autoHideIdleTimeout` | Idle timeout (seconds, 0 to disable) | `0` |
| `autoHidePlus.excludeLanguages` | List of excluded languages | `[]` |
| `autoHidePlus.hideOnFocusLost` | Hide when VS Code loses focus | `false` |
