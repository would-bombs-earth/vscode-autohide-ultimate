# Change Log

## 2.0.3 (Ultimate Edition / 终极版)
- **Bilingual Documentation**: Merged Chinese and English READMEs for better accessibility. / **双语文档**: 合并中英文 README，确保详情页清晰可见。

## 2.0.0 (Ultimate Features / 终极版核心功能)
- **Global Master Switch**: Quickly enable/disable all features. / **全局总开关**: 一键启用/禁用所有功能。
- **Hide Delay**: Added precision control for smooth transitions. / **隐藏延迟**: 增加精准延迟控制，动画更平滑。
- **Exclusion Patterns**: Prevent auto-hide on specific languages. / **语言排除**: 针对特定编程语言禁用自动隐藏。
- **Idle Auto-Hide**: Automatically clean UI after inactivity. / **闲置自动隐藏**: 停止操作后自动清理界面。
- **Focus-Lost Hide**: Hide UI when switching away from VS Code. / **失去焦点隐藏**: 切换应用时自动收起侧边栏。
