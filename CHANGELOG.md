# Change Log
 
## 2.0.0 (Ultimate Edition)
*   **Global Master Switch**: Quickly enable/disable all features via a single toggle.
*   **Hide Delay**: Added precision control for smooth UI transitions.
*   **Exclusion Patterns**: Prevent auto-hide on specific languages (e.g., Markdown).
*   **Idle Auto-Hide**: Automatically clean UI after a period of inactivity.
*   **Focus-Lost Hide**: Automatically hide UI when switching away from VS Code.
*   **Multi-language Support**: Full localization for 11 world languages.
*   **Brand Update**: Rebranded to "Auto Hide Ultimate" by Ultimate Studio Team.
